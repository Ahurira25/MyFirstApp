Introduction

HTML stands for HyperText Markup Language.It describes the structure of webpages and this is achieved by adding code to the words you want to appear on the page.

CSS stands for Cascading Style Sheet.It uses rules to enable you control the styling and layout of webpages.There are a variety of CSS propertiesyou can use in your CSS rules.

  • To create a webpage, you start by writing down the words you want to appear on your page.Then you add tags or elements to the words so that the browser knows what is a heading and what is a paragraph.
  • CSS properties fall into two categories;
    • Presentation:how to control things like how to add the background to images,background colors to images,color of text,the fonts you want to use and the size of those fonts.
    • Layout:how to controlwhere the different elements are positioned on the screen.

HTML and CSS

HTML and CSS are different in this way;

  • HTML code is made up of characters that live inside angeled brackets and these are called HTML elements.
  • Elements are usually made up of two tags opening tag and closing tag
  • CSS makes a page look more attractive through adding color,font-familly and font size to the content within the tags or elements
